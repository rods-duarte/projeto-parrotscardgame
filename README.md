# Parrots card game

A themed memory game with randomness, effects and transition, responsiveness and dancing parrots!

<br>

# Features
<ul>
<li> Select a number of pairs </li>
<li> Cards are randomly put on the table </li>
<li> turning cards animation </li>
<li> Time counter </li>
<li> Moves counter </li>
</ul>
